# Editor de reportes

Su objetivo es facilitar la elaboración de reportes a partir de una plantilla.

## Instalación

    npm install

## uso
    node server.js

Luego:
- Navegar a http://localhost:3000
- Cargar la plantilla
- Editar el contenido
- Opcionalmente, cambiar a vista de código, editar y volver a vista WYSIWYG
- Guardar
- Ver
